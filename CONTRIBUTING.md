# Contributing to Awesome Self-Hosted VPS

Thanks for your interest in contributing! This list aims to be a high-quality, community-maintained resource for self-hosting enthusiasts.

## How to Contribute

1. Fork this repository
2. Create a new branch for your addition
3. Make your changes following the guidelines below
4. Submit a pull request

## Guidelines

### Adding a Tool

- **One tool per PR** — keep changes focused and reviewable
- **Actively maintained** — the tool must have a commit within the last 12 months
- **Description format** — explain *who it's for* and *what problem it solves*, not just features
- **Honest descriptions** — no marketing copy; mention limitations where relevant
- **Correct license** — verify the license from the repository, not from memory

### Adding a VPS Provider

- **Real pricing** — check the actual pricing page, include the date you last verified
- **Honest notes** — mention both strengths and limitations
- **Personal experience preferred** — if you've actually used the provider, mention it

### Format

Use this format for tool entries:

```markdown
| [Tool Name](https://tool-website.com) | Short description — what it does and who it's for | License | [GitHub](https://github.com/org/repo) |
```

### What We Don't Accept

- Tools that are abandoned or archived
- Duplicate entries (check existing entries first)
- Self-promotion without disclosure
- Entries with affiliate links

## Code of Conduct

Be respectful, constructive, and helpful. We're all here to make self-hosting more accessible.
